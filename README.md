# pixel-art-logo-maker

Black and white pixel art logo maker built with Grok.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
```

Output is written to `dist/`.

## Features

- Square grid editor (5×5, 8×8, 10×10, 16×16, or custom size)
- Draw / erase with mouse or keyboard
- Random symmetrical seed patterns with configurable mirror modes
- Export fixed 512×512 centered PNG logos
